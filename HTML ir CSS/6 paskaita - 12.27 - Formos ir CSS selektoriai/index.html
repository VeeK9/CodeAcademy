FORMOS

Formos tag'ai

  • form - forma
  • label - etikete ivesties laukui
  • input - ivairus ivesties laukas
  • textarea - teksto ivesties laukas (didelio kiekie)
  • select - pasirinkimo variantai (mazai)
  • datalist - pasirinkimo variantai (daug)
  • option - pasirinkimo variantas

Formos atributai

  • for - etiketes atributas skirtas ja sujungti su ivestimi
  • id - label sujungimas su ivestimi (turi buti toks pats tekstas kaip ir for)
  • name - priskiria ivesciai varda (dazniausiai toks pat kaip id)
  • type - skirtas nurodyti input'o paskirti.
    • text - paprastas tekstas
    • password - slaptazodis (slepiamas tekstas)
    • email - elektroninis pastas
    • number - skaicius
    • submit - formos pateikimo ivestis
    • url - nuoroda
    • file - failas is irenginio
    • checkbox - multi variantu pasirinikimas (privalo tureti vienodas name atributu reiksmes ir skirtingas value atributu reiksmes)
    • radio - vieno is keletos variantu pasirinkimas (privalo tureti vienodas name atributu reiksmes ir skirtingas value atributu reiksmes
    • date - datos pasirinkimas (metu, menesio, dienos)
    • datetime-local - datos ir laiko pasirinkimas (metu, menesio, dienos, valandu ir minuciu)
    • month - menesio pasirinkimas
    • week - savaites pasirinkimas
    • time - laiko pasirinkimas
    • color - spalvos pasirinkimas
    • button - mygtukas be funkcionalumo
    • submit - formos pateikimo input'as mygtuko pavidalu
    • image - formos pateikimo input'as paveiksliuko pavidalu
    • hidden - paslepia ivesti, bet reiksme yra pateikiama
    • range - slenkancios juostos skaiciaus pasirinkimas
    • reset - mygtukas isvalantis formos ivestis
    • search - paieskos laukas
    • tel - telefono ivesties laukas
  • Required - privalomas ivesties laukas (be jo neleis pateikti formos)
  • min - minimali reiksme (skaiciaus)
  • max - maximali reiksme (skaiciaus)
  • step - zingsnis (skaiciaus)
  • type - failo tipas ar tipai, kuriuos leidziama pateikti
  • pattern - kokia seka kazkas turi buti parasyta (regex :))
  • placeholder - tekstas atvaizduojamas input'o arba textarea viduje
  • checked - is ankasto pazymetas radio arba checkbox
  • cols - textarea dydzio nustatymai
  • rows - textarea dydzio nustatymai
  • value - ivesties reiksme
  • selected - is anksto parinktas option'as
  • disabled - neleidziamas pasirinkti option'as

Selectors

  • Atributai
    • [atributas] - atributas
    • [atributas="reiksme"] - atributas su specifine reiksme
    • [atributas~="reiksme"] - atributas su zodziu reiksmeje
    • [atributas*="reiksme"] - ... zodzio dalimi reiksmeje
    • [atributas|="reiksme"] - ... lygus "reiksme" arba prasidedantis "reiksme-"
    • [atributas^="reiksme"] - ... prasideda "reiksme"
    • [atributas$="reiksme"] - ... pasibaigia "reiksme"
  • Pseudo klases formoms
    • input:checked - ar input'as pazymetas
    • input:default - pradinis pazymetas input'as
    • input:disabled - formos elementas turintis disabled atributa
    • input:enambled - formos elementas neturintis disabled atributo
    • input:focus - siuo metu aktyvus input'as
    • input:in-range - ivestis ribose
    • input:out-of-range - ivestis uz ribu
    • input:indeterminate - nenustatyta reiksme (nesvarbus)
    • input:valid - teisinga ivestis
    • input:invalid - neteisinga ivestis
    • input:optional - neturi required atributo
    • input:required - turi required atributa
    • input::placeholder - galimybe redaguoti placeholder teksta
    • input:read-only - turintis readonly atributa
    • input:read-write - neturintis readonly atributo
  • Pseudo elementai
    • ::before - sukuria turini pries elementa
    • ::after - sukuria turini uz elemento
    • ::first-letter - pirmoji elemento raide
    • ::first-line - pirmoji elemento teksto eilute
    • ::marker - rikiuotu ar nerikiuotu saraso elementu zymes
    • ::selection - pazymeto turinio stilius
    • ::-webkit...
    • ::-webkit-scrollbar...
