Libraries ir Frameworks

CSS Frameworks/Libraries - Kazkieno kito parasytas CSS kodas, kuri tu gali naudoti savo projectuose. Norint naudotis framework'ureikia ji prijungti prie savo projekto naudojant CDN (content delivery network). Didzioji dauguma CSS framework'u veikia tokiu principu - prijungus prie savo projekto gali naudoti ju viduje aprasytas klases - priskirineti HTML elementams framework'o klases, o pats gali net nerasyti jokio CSS. Yra classless CSS framework'u. Juos naudojant, nereikia suteikti HTMl elementams klasiu. Viskas yra aprasyta naudojant tag'u/nth/attributu... selektorius.

Pros

  • Pagreitina/palengvina darba
  • Nereikia/labai mazai rasyti CSS
  • Lengva naudotis
  • Idiegtas responsiveness
  • ...

Cons

  • Daug perteklinio/nenaudojamo kodo

  • Nesvarus HTML del klasiu pertekliaus
  • Neunikalus dizainas
  • Prastesnis google paieskos sistemos (SEO) ivertinimas
  • Pasimirsta CSS

Preprocesors ir Units (dydzio vienetai)

CSS with superpowers

Suteikia galimybe rasyti CSS'a naujoviskesniu budu, kur galima lengvai kurti kintamuosius, nest'inti selektorius, rasyti: salygas, sunkcijas, mix'inus ir t.t.

Naudojant betkuri CSS preprocesoriu, reikia parasyta koda konvertuoti i paprasta CSS'a, kuri supras svetaine.

  • SASS/SCSS
  • LESS
  • Stylus

  • ...

Units (dydzio vienetai)

  • px
  • % - tevinio elemento dydi
  • vh - viewport height - matomos svetaines dalies aukstis (100 - visas aukstis)
  • vw - viewport width - matomos svetaines dalies plotis (100 - visas aukstis)
  • rem - root em - 16px - sakninius nustatymus
  • em - 16px - teinio elemento nustatymus

Git ir Github

Git - programa, kuri versijuoja koda.

Github - Svetaine, kurioje keliamas (versijuojamas) kodas.

Git ir Github setup
  1. Isirasyti Git'a i kompiuteri.
  2. Pasitikrinti ar tikrai isirase Git'as. terminale panaudoti komanda "git -v"
  3. Susikurti Github paskyra.
  4. Konfiguruoti git'a globaliai (visame irenginyje)
    • git config --global user.name GitHubUserName
    • git config --global user.email GitHubEmail
    • git config --global core.editor "code"
    • git config --global core.autcrlf input
  5. Patikrinti ar teisingai config'inta. terminale panaudoti komanda "git config --global --edit"
