Transformacija

Transformacija nurodo kaip kinta elementas. Elementas gali kisti 3-5 budais.

  • translate - judina elementa nurodytoje asyje.
    • translate(x, y)
    • translate3d(x, y, z)
    • translateX(x)
    • translateY(y)
    • translateZ(z)
  • rotate - rotuoja
    • rotate(z)
    • rotate3d(x, y, z, a)
    • rotateX(x)
    • rotateY(y)
    • rotateZ(z)
  • scale - keicia dydi / tempia/spaudzia
    • scale(x, y)
    • scale3d(x, y, z)
    • scaleX(x)
    • scaleY(y)
    • scaleZ(z)
  • skew - tempo kampus
    • skew(x, y)
    • skewX(x)
    • skewY(y)
  • matrix - universalus
  • matrix3d - universalus
  • perspective - perspektyvos keitimas

Kartu su transform dar gali buti naudojamas ir transform-origin, kuris nusako transformacijos pradini taska. Default yra centras.

Tranzicija

  • transition-duration - per kiek laiko turi ivykti pasikeitimas
  • transition-timing-function - kaip pasikeitimo laikas yra paskirstytas (default - ease)
  • transition-property - nurodo, kuris stilius turi pasikeisti (default - all)
  • transition-delay - nurodo po kiek laiko prasides kitimas
  • transition - visu junginys (duration, delay, (kiti nesvarbu))

Animacija

Animacijos nustatymai

  • animation-name - animacijai suteiktas vardas
  • animation-duration - animacijos trukme
  • animation-iteration-count - kiek kartu vyks animacija (default 1)
  • animation-play-state - animacijos busena (vykdoma ar nevykdoma)
  • animation-timing-function - kaip pasikeitimo laikas yra paskirstytas (default ease)
  • animation-direction - kryptis
  • animation-delay - animacijos uzdelsimas
  • animation-fill-mode - ka daryti pries/po animacijos
  • animation-composition - ka datyti su stiliais, kurie turi kelis nustatymus
  • animation - apjungimas (duration, delay, (kiti nesvarbu))

Animacijos aprasymas (veiksmai / zingsniai)

Animacijos zingsniaiaprasomi naudojant keyframes. Keyframes gali buti nurodomi specifiniais zodziais arba procentas. Nuo 0 iki 100. From -> To. Kiek procentu zingsniui yra skiriama - tiek apytiksliai laiko is duration'o ir bus suteikiama tam zingsniui atlikti. 

        @keyframes animation-name{
          0%{
          } x% {
          } y% {
          } 100% {
          }
        }
