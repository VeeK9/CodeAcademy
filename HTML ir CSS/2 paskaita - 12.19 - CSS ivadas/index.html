CSS selektoriai

  • * - visko selektorius
  • tagName - tag'o selektorius
  • .className - klases selektorius
  • #idName - id selektorius

* - parenka visiskai visus elementus

tagName - parenka visus elementus su nurodytu tag'o pavadinimu

Klase

.className - parenka visus elementus, kurie turi klases atributa su tokiu paciu vardu

Viena klase gali buti priskirta daugeliui elementu.

Vienas elementas, gali tureti daugeli klasiu.

ID

#idName - parenka visus elementus, kurie turi id atributa su tokiu paciu vardu

Vienas ID, gali buti priskirtas tik vienam elementui.

Vienas elementas gali tureti TIK VIENA ID.

Pavadinimu kurimas

  • camelCase - kiekvienas naujas zodis prasideda is didziosios raides.
  • snake_case - kiekvienas naujas zodis prasideda uz apatinio bruksnelio.
  • kebab-case - kiekvienas naujas zodis prasideda uz paprasto bruksnelio

Colors

  • name
  • rgb
  • hex
  • hsl

Name - nurodomas spalvos vardas. Spalvu vardu yra apie 140.

RGB - red green blue

rgb(red, green, blue) spalvos kiekis nurodomas nuo 0 iki 255

rgba(red, green, blue, alpha) - rgb nuo 0 iki 255; o alpha nuo 0 iki 1, 0.001 zingsniu. - permatomumas rgb

Maksimalus spalvu kiek, neskaitant alpha, yra 256*256*256=16777216 spalvu

Hex - hexdecimal - sesioliktainis

hex spalvos kiekis nurodomas nuo 0 iki f

Inline

  • span
  • input
  • label
  • textarea

Linijiniai elementai negali tureti virsutinio ir apatinio margin'o (atitraukimas nuo aplinkiniu elementu). Linijiniai elementai negali tureti plocio ir aukscio nustatymu.

Positions

  • static
  • relative
  • absolute
  • fixed
  • sticky

Suteikus elementui bet kuria apart static pozicijos, jis igauna galimybe naudotis siais stiliais: z-index, top, right, bottom, left;

z-index nusako elemento eiles sluoksni ekrane (kaip arti/toli nuo vartotojo yra elementas (kuris elementas kuri uzstos))

Static

Statix nustatymai nesuteikia jokiu ypatybiu. Net galimybes naudotis: z-index, top, right, bottom, left;

Relative

Relative nustatymas suteikia elementui galimybe naudotis: Z-index, top, right, bottom, left nustatymais.
Top, right, bottom, left nustatymai pastumia elementa nuo nurodytos krastines per nurodyta atstuma nuo elemento originalios pozicijos.
Elemento viduje esantis absolute pozicijos elementas, atsitraukia nuo elemento krastu jei sis turi relative pozicija.

Absolute

Absolute nustatymas suteikia elementui galimybe naudotis: Z-index, top, right, bottom, left nustatymais.
Top, right, bottom, left nustatymai nurodo per koki atstuma elementas atsitrauks nuo artimiausia protevinio elemento, turincio relative pozicija, ribu.
Elementas turintis absolute pozicija, iseina is puslapio tekmes (jo pozicijos neitakoja puslapyje issideste elementai).

Fixed

Fixed nustatymas suteikia elementui galimybe naudotis: Z-index, top, right, bottom, left nustatymais.

Elementas su fixed pozicija iseina is puslapio tekmes.
Fixed elementas uzsifiksuoja vienoje ekrano vietoje ir kad ir kiek butu scroll'inama, jis juda kartu.
Top, right, bottom, left nustatymai nurodo per koki atstum, nuo kuriuos viewport krastines jis atsitraukia ir uzsifiksuoja.

Sticky nustatymas suteikia elementui galimybe naudotis: Z-index, top, right, bottom, left nustatymais.

Sticky pozicija turintis elementas prilimpa prie ekrano (viewport) krastu per nurodyta atstuma.
atkimba tik pasiekes tevinio elemento ribas arba grizes i savo originalia pozicija.
