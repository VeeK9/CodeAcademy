Media Queries

Media Queries yra naudojami norint svetainei suteikti skirtingus stilius, esant skirtingiems ekranu dydziams
CSS'e yra rasoma @media (min arba max -width: ...px){ ribos kur rasomas CSS tam ekrano dydziui}

Media Queries ekrano luzio taskai (breakpoints)

  • no media mazesniam 576px - telefonai portreto pozicijoje
  • didesniam negu 576px - telefonai peizaziniai
  • didesniam negu 768px - plancetes
  • didesniam negu 992px - mazi desktop (laptop)
  • didesniam negu 1200px - desktop
  • didesniam negu 1400px - large desktop

No media, kodel?

Pradiniai stiliai turi buti rasomi be jokio media query. nes Media query dazniausiai pakeicia tik elementu isdestyma arba ju atvaizdavima/neatvaizdavima, bet ne kazkokius bendrus stilius (fono spalva, remelius, teksto spalva, srifta...). Bendri stiliai yra taikomi visiems ekranu dydziams, tai kad ji nekartoti kiekviename media query - juos aprasai be media query ir tuomet jie taikomi visiems ekranu dydziams.
