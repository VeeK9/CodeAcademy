JSON server

JSON serveris yra NPM (Node Package Manager) paketas, kurio pagalba galima susikurti lokali API serveri su savo sugalvotais duomenimis, kurie privalo buti JSON faile, JSON formatu. I si serveri galima bus siusti fetch uzklausa su ivariais metodais.

Tam, kad galetume naudotis JSON serveriu mums reikia:

  1. npm (isiraso kartu su Node'u)
  2. Isirasyti json-server globaliai
  3. Susikurti JSON faila tinkama struktura
  4. pasileisti JSON faila su json-server komanda

Terminalo komandos

  • cd - change directory - vaikscioti tarp aplankalu
  • ls - list files - isvardina failus/aplankalus esamoje direktorijoje (mac/linux)
  • dir - tik windows, tas pats kas ls
  • json-server filename.json
  • clear - isvalo terminalo ekrana (nukelia aukstyn)

Navigavimas (cd +?)

  • / - projekto.sistemos pradzia
  • ./ - niekur (lieki toje pacioje vietoje)
  • ../ - vienas zingsnis per aplankala atgal

JSON server Routes

*Kiekvienas duomenu laukas (dName) gauna savo route'us*

  • GET dName - gauti visus dName lauke esancius duomenis
  • GET dName/id - gauti dName lauke esanti duomeni su nurodytu id
  • POST dName - papildyti dName lauke esancius duomenis nauju duomenimi
  • PUT dName/id - redaguoja dName lauke specifini duomeni, viska (isskyrus ID) perrasydamas nauja info
  • PATCH dName/id - redaguoti specifini duomeni (pagal id) dName lauke, bet tik ta informacijos dali, kuri buvo paduota
  • DELETE dName/id - trinimas specifinis duomuo (pagal id) dName lauke
