Moduliai

Moduliai yra atskiri kodo fragmentai isskaidyti skirtinguose failuose.

MOduliuose gali buti ivairus kodas. Nuo duomenu (kintamuju) iki funkciju, Klasiu, metodu ir kt.
JS Klases turi buti atskiri failai. Kitaip pasakius - viename faile neturetu buti daugiau negu vina JS Klase.
HTML'e prijungtas script'as, kuris kreipiasi i kitus script failus, privalo tureti nurodyta tipa - modulis (type='module')
HTML failas, kuriama yra modulio tipo JS failas, privalo buti paleistas kaip serveris, kitu atveju neleis vykdyti JS failo kodo.

Export ir Import

Failuose esanti informacija gali buti iskeliama (export'uojama) ir galima kazkokia informacija isikelti (import'uoti) is kito failo

Export

Informacija galima exportuoti dviem budais:

  • export default - numatytasis/pagrindinis export'as. Jis gali buti TIK VIENAS viename faile.
  • export - paprastas/papildomas export'as/ai, kuriu gali buti daug viename faile. Jie visi gali buti israsyti vienoje eiluteje.

Import

Informacija gali buti importuojama ivairiais budais.
Bet yra keletas pagrindiniu budu:

  • import RandomDefaultName from './.../x*.js' - kai yra importuojamas default export'as galima jo pavadinima pasirasyti betkoki.
  • import {sameName as diffName} from './.../*.js' - yra importuojamas paprastas export'as kitokiu pavadinimu.
  • import DefaultStuff, {nonDefaultStuff} from './.../*.js' - viename import aprasyme galima ipmportuoti ir default ir paprastus exportus.
